# AROS — Autonomous Research Operating System

A domain-agnostic autonomous research platform designed to improve measurable systems through iterative experimentation.

## Philosophy

The system separates research, optimization, execution, and evaluation. No single component is trusted to judge its own success.

- Research proposals are generated by an AI Research Agent
- Numerical optimization is performed by dedicated search algorithms
- Evaluation is performed independently
- All experiments are recorded permanently

This separation minimizes bias, improves reproducibility, and prevents self-reinforcing failure modes.

## Architecture

```
Research Agent → Hypothesis Generator → Search Engine → Experiment Runner → Evaluator → Experiment Database → Research Agent (loop)
```

## Getting Started

```bash
pip install -e .

```
